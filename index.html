How does it work?

The script accepts dummy details for username and password fields to log into X.
The following details are automatically captured in screenshot AND scraped-texts for the supplied scrape_username:-
  1. Account Timeline
  2. Account Bio
  3. A certain number of posts (max_tweets)
  4. Followers
  5. Following

What this does?

The credentials.py file contains certain inputs that the tool makes use of. A chronological sequence of events is illustrated below :--

  1. Initiating the process - opening Instagram using android package.
  2. Waiting for login screen to load.
    • if account present? automatically logs in.
    • else fetches EMAIL and PASSWORD variables from the credentials.py file and logs in using those details.
  3. Closing any automatic pop-up windows.
  4. Opeing IG dm and searching for username using TARGET_USERACCOUNT_FOR_MESSAGE variable inside credentials.py file.
  5. Waiting for screen to load(The automation would break otherwise in absence of a suitable wait strategy)
  6. Opening the chat of the username provided above and scrolling till a specified date in time, using DATE variable provided by credentials.py file
  7. Scrolling down while taking screenshots of chat conversation until the tool finds today`s date.
  8. Generating pdf from the captured screenshots in a customized fashion (here, 4 screenshots in one sheet)
  9. Preparing form 65B &case-report front page
  10. Ending the task

PDF 4/5

PDF: Certificate Of Integrity

In investigative filings by the National Investigative Agency (NIA), a Certificate under Section 65B of the Indian Evidence Act, 1872 is crucial for the admissibility of electronic evidence, such as screenshots, in court.
This certificate authenticates the electronic record, confirming that it was produced by a reliable computer system and has not been tampered with.
It ensures that the evidence meets the legal standards for accuracy and integrity, thereby strengthening the credibility of the evidence presented. Without this certification, electronic evidence may be deemed inadmissible, potentially undermining the prosecution’s case.

The certificate can be customized with precise language suggested by NIA and can be generated by the tool along with the screenshots document.
